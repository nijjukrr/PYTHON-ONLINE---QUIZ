# 🧬 QuizForge: Neural Assessment Portal

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)
[![Streamlit](https://img.shields.io/badge/Streamlit-1.25+-FF4B4B.svg)](https://streamlit.io/)
[![Design](https://img.shields.io/badge/Design-High--Fidelity-emerald.svg)](#)

QuizForge is a premium, cinematic online assessment platform designed for a high-performance developer experience. Featuring a high-contrast "Neural" aesthetic, real-time analytics, and a seamless 3D-integrated landing experience.

## ✨ Key Features

- **🚀 Cinematic Landing**: Integrated 3D Robotic interface with smooth transitions to the neural hub.
- **⚡ High-Precision Quiz Engine**: Rounded option cards with instant feedback markers and a dynamic neural timer.
- **📊 Performance Synchronization**: Custom results dashboard featuring circular points charts and a 2x2 metric grid (Correct, Wrong, Skipped, Max Pts).
- **📜 Neural Logs**: Robust historical tracking system that harmonizes legacy and new assessment data.
- **⚙️ Admin Override**: Full-featured admin portal for question bank management with emerald-gradient action buttons.
- **🏆 Global Rankings**: Real-time leaderboard with multi-domain filtering.

## 🛠️ Technology Stack

- **Core**: Python 3.9+
- **Frontend**: Streamlit (Multi-page Architecture)
- **Styling**: Custom CSS (Neural Glassmorphism, Space-to-Surface Gradients)
- **Data**: JSON (Questions), CSV (Analytics)

## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/nijju-kr/python-online-quiz.git
cd python-online-quiz
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Launch the Portal
```bash
python -m streamlit run app.py
```
